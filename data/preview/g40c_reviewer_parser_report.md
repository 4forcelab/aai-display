## 9. Methodology Note
G40C parses the source-backed reviewer table stored in the raw API field 最有利標:評選委員.
The field is represented as a Python literal string containing reviewer dictionaries, so ast.literal_eval is used for parsing.
Reviewer records are extracted only when reviewer identity can be safely separated from table structure.
Case-level summary joins G40B enrichment fields by case_number for arrangement and filtering.
